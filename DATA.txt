CupCast / FIFAproject2026 Data

The full data folder is not included in this GitHub repository.

Download the full data package from Google Drive:

https://drive.google.com/file/d/1bJnvtBXX1PASxo_KnYHa7Js0-cYG1v1I/view?usp=drive_link

After downloading, place the extracted data folder at:

data/

Expected local paths include:

data/raw/
data/processed/
data/external/
data/interim/

Large generated outputs may also be restored locally under:

output/simulations/
output/brackets/
output/diagnostics/
