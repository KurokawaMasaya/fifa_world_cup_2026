# Live Folder

Current files to use:

- `fixtures_results.csv` - normalized ESPN fixtures/results.
- `group_standings.csv` - normalized ESPN standings if available.
- `worldcup_group_stage_live_summary.csv` - live prediction evaluation summary.
- `worldcup_group_stage_live_evaluation.csv` - match-level live evaluation.
- `live_tournament_simulation.csv` - current state-conditioned live tournament simulation.
- `worldcup_group_stage_actual_results.csv` - manual/actual result template.

Human-readable log files are under `logs/`.
Raw ESPN JSON is under `raw/`.
